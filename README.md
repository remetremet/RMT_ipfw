# RMT_ipfw
Automated FreeBSD IPFW2 (up to 10xWAN, up to 10xLAN, dualstack, router, endpoint, VPN)

- Statefull/Stateless firewall
- Multiple WANs (FreeBSD FIBs)
- Multiple LANs (incl. VLANs)
- IPv6/IPv4
- Host and port based
- Failover / Load balancing of multiple WANs


Requirements
---
- `bash`
- FreeBSD IPFW2
